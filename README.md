#  Application de Sondages en Ligne

Application web développée avec Django permettant de créer, partager et analyser des sondages.

---

##  Fonctionnalités
- Création de sondages
- Partage via lien
- Collecte des réponses
- Visualisation des résultats

---

##  Technologies
- Django (MVT)
- MySQL
- JavaScript
- Chart.js
- Bootstrap
